# Modern Thames Blueprint Theme

This document outlines the visual identity for the CityPee application, codenamed "Modern Thames Blueprint." It provides a canonical reference for colors, typography, spacing, and radii to ensure a consistent and professional user experience.

## Color Palette

The palette is designed for high contrast and accessibility, inspired by the functional aesthetic of architectural blueprints and modern London infrastructure.
